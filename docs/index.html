01 — 保持唤醒

一个开关,
整台 Mac 不再休眠

基于 IOPMAssertion 向系统声明「请勿空闲休眠」,无论插电还是电池,通通有效

Prevents idle system sleep · on AC & battery

02 — 屏幕策略

系统醒着,
屏幕由你决定

允许息屏省电,或让屏幕常亮,系统始终不休眠,屏幕亮不亮你说了算

Allow display sleep · or keep it always on

03 — 保持时长

设定时长,
到点自动收手

一直保持,或 30 分钟 / 1 / 2 / 5 小时,计时结束,自动交还系统默认策略

Keep for a set duration · auto-restores

04 — 合盖继续运行

合上盖子,
任务继续跑

合盖也不休眠,下载、渲染、跑服务统统不中断,贴心提醒散热与电量,建议插电使用

Runs with the lid closed

05 — 切换时免输密码

开合之间,
不再反复输密码

一次性授权后,合盖、开盖切换无需每次输入密码,顺手又安心

One-time authorization · skip the wake password

06 — 开机自启 · 原生

登录即就位,
像系统自带一样

SMAppService 注册登录项,开机自动待命,系统设置风格、系统强调色、SF Symbols、菜单栏常驻

Launch at login · native to macOS
