# 更新日志

本项目遵循语义化版本（SemVer）。

## [1.0.0] - 2026-05-31

首个发布版本。

### 新增
- 保持唤醒：基于 IOPMAssertion 阻止系统空闲休眠，插电与电池均生效。
- 屏幕策略：可选「允许屏幕关闭」（省电）或「屏幕常亮」。
- 定时自动关闭：一直 / 30 分钟 / 1 / 2 / 5 小时，到时自动恢复。
- 合盖继续运行：通过 `pmset disablesleep` 让合盖也不休眠。
- 合盖免密：一次性写入精确限定的 `/etc/sudoers.d` 规则，之后切换免输密码；未配置时自动回退到管理员授权弹窗。
- 开机自启：基于 `SMAppService`。
- 原生 macOS 系统设置风格界面，常驻菜单栏，无 Dock 图标。
- 多分辨率应用图标。

### 说明
- 当前版本为 ad-hoc 自签、未做 Apple 公证；首次打开需在「隐私与安全性」放行一次。
- Apple Silicon 合盖休眠部分由硬件门控，建议插电 / 接外接显示器时使用最稳。
